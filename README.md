# SimpleMMO Bot Documentation

This repository hosts the public documentation website for **SimpleMMO Bot**, a licensed, Telegram-controlled automation companion for SimpleMMO.

## Website

Visit [topup.eu.org](https://topup.eu.org) for product information, setup instructions, feature documentation, licensing details, and troubleshooting.

## Public contents

- User-facing documentation
- Configuration template guidance
- Feature and API reference
- Release and installation instructions
- Security and privacy information

The production Python source, license server administration tools, private signing keys, user credentials, and compiled build workspace are not included in this public repository.

## Security

Never publish `.env`, game cookies, Telegram bot tokens, passwords, license keys, CSRF tokens, API tokens, or private keys in a GitHub issue.

## Disclaimer

Automation may violate SimpleMMO's terms or rules and may result in account restrictions. Users are responsible for reviewing applicable third-party terms and for all activity performed with their accounts.

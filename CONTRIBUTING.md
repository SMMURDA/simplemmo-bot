# Contributing to the Documentation

This public repository accepts documentation corrections, accessibility improvements, and reproducible issue reports.

## Before opening an issue

- Use the latest release.
- Read the troubleshooting guide.
- Remove every credential from screenshots and logs.
- Include the app version and operating system.

## Never submit

- Completed `.env` files.
- SimpleMMO cookies, passwords, CSRF tokens, or API tokens.
- Telegram bot tokens.
- Full license keys.
- Private signing keys or license administration credentials.

The production application source and license service administration are maintained privately and are not accepted through this repository.

# Portal v2 Deployment

The complete deployment and administration guide is included in the private source package and the combined ZIP as `START-HERE-PORTAL-V2.md`.

Deploy order:

1. Back up D1.
2. Apply `0004_payment_methods_currency.sql`.
3. Deploy the Worker.
4. Deploy this GitHub Pages folder.
5. Configure payment methods at `/admin/payment-methods/`.
